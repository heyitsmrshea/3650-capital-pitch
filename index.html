About this project

Each design direction uses the same 3650 Capital content and photography from the existing site. The goal is to demonstrate range — six fundamentally different approaches to the same brand, each with its own design system, typography pairing, color architecture, and layout philosophy.

Built with Claude AI + MCP browser integration + Stitch design references. Every variant includes scroll animations, counter effects, responsive layouts, and accessibility considerations.
